배열요소의 위치 찾기
indexOf()
lastIndexOf



배열요소의 추가/삭제
push(a,b,c,...) : 마지막에 요소를 추가
pop() : 마지막 요소를 제거하여 리턴



배열요소의 추출
slice(start, end) : start ~ end -1까지 추출한다.



배열의 수정 - 삽입, 삭제
splice(index, n, a, b, c....) : index위치에서 n개를 삭제하고 a, b, c아이템을 삽입한다.
n=0이면 삭제없이 아이템을 추가한다.
splice(index, n) index위치에서 n개를 삭제한다.



배열의 문자열 변환
join(구분자) : 구분자를 지정한다., 구분자를 생략하면 ,를 기준으로 한다.
