Position : static 기본값, 코딩 순서대로 배치

Position : relative 상대배치, static을 기준으로 재배치

position: relative;
top: 150px;
left: 100px;

Position : absolute 절대배치, container를 기준으로 재배치

position: absolute;
top : 200px;
left : 150px;

Position : fixed 고정배치, container를 기준으로 재배치

position: fixed;
top : 300px;
right: 100px;
